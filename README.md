# Aditya Pathak — Portfolio Website (Static)

A modern, responsive portfolio for **Performance Marketing Specialist** roles.

## Files
- `index.html`
- `styles.css`
- `script.js`

## How to run locally
Just open `index.html` in a browser.

## Deploy (easy options)
### 1) Netlify (drag & drop)
- Zip the folder `portfolio-aditya/`
- Go to Netlify → *Add new site* → *Deploy manually* → drag the zip

### 2) GitHub Pages
- Create a repo (e.g., `aditya-portfolio`)
- Upload these files to the repo root
- Settings → Pages → Deploy from branch

## Resume download button
In `index.html` hero section, the **Download Resume** button points to `resume.pdf`.

Options:
- Put your PDF in this folder as `resume.pdf` (recommended)
- OR replace the link with a public Google Drive/Dropbox URL.

## Make it stronger (recommended)
Add links + metrics to each project card:
- CTR, CPC, CPA, ROAS, CVR
- screenshots (campaign structure, GA4 funnels, creatives)

If you send screenshots/metrics, the projects can be upgraded into full case-study pages.
